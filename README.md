## How do I get RTVS?

- You can [download RTVS from here](https://aka.ms/rtvs-current). 
- You can [read the documentation for RTVS here](https://microsoft.github.io/RTVS-current)
- You can [read the source code for RTVS here](https://github.com/Microsoft/RTVS)

## How do I get Visual Studio?

If you are a student, an academic, someone working at a small business, you are entitled
to [download and use Visual Studio Community edition for free](https://www.visualstudio.com/en-us/products/visual-studio-community-vs.aspx).

You can find out more about the details of the Visual Studio Community Edition license from
[the web site](https://www.visualstudio.com/support/legal/mt171547).
